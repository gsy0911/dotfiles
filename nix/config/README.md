# keybind policy

# zsh

`CTRL|{}` = `control + {}`

# neovim

`<Leader>{}` = `space + {}`
or
`CTRL|{}` = `control + {}`

# wezterm


`CMD|{}` = `command+ {}`
or
`CMD|SHIFT|{}` = `command + shift + {}`

# aerospace

`ALT{}` = `option + {}`
or
`ALT|SHIFT{}` = `option + shift + {}`


# zellij

default `CTRL|{}` base
